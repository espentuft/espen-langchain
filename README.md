# Espen LangChain

A project for experimenting with and building applications using LangChain.

## Getting Started

This project is a starting point for LangChain development.

## Setup

```bash
# Install dependencies
npm install  # or pip install if using Python

# Run development server
npm run dev
```

## Project Structure

- `/src` - Source code
- `/examples` - Example implementations
- `/docs` - Documentation

## Technologies

- LangChain
- Node.js / Python (choose one)
- TypeScript (optional but recommended)
